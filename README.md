XYZ Bookstore
Einfache responsive Bookstore-Landingpage mit HTML&CSS.
# Live Demo
https://mark15x.github.io/Bookstore-Page/
# Features
-Responsive Layout
-Buchkarten mit Bildern
-Gestylt Navigation
-Call-to-Action Buttons
-Modernes,klares Design
# Verwendete Technologien
-HTML5
-CSS3
-Flexbox
# Lernziel
Dieses Projekt ist Teil meines Lernwegs zum Fachinformatiker für Anwendungsentwicklung.
Der Fokus lag auf Layout-Übung,Card-Design und sauberer Struktur.
